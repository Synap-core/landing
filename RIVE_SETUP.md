# Getting Your Rive Animation

## Quick Start - Download a Free Animation

1. **Visit Rive Community**: https://rive.app/community/

2. **Search for:**
   - "abstract background"
   - "particle system"
   - "neural network"
   - "tech background"

3. **Download Steps:**
   - Click on an animation you like
   - Click "Open in Editor"
   - Click "Share" icon (top right)
   - Select "Download â†’ For newest runtime"
   - Save as `hero-bg.riv`

4. **Add to Project:**
   - Move `hero-bg.riv` to `/public/animations/` folder
   - Restart dev server

## Recommended Animations

## Customization

Once you have a .riv file:
1. Open in Rive Editor
2. Change colors to match brand (#10B981, #3B82F6)
3. Adjust animation speed
4. Re-export
